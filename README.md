# Projeto 1 - Funções de Espalhamento (hashing)
Repositório destinado ao projeto 1 da disciplina de Algoritmos e Estruturas de Dados II da Universidade Federal do ABC (UFABC) no quadrimestre suplementar 2021.1.

## Arquivos
O projeto possue os seguintes arquivos:

- `hashing.h`: arquivo de cabeçalho que declara as funções de espalhamento para os métodos de divisão e multiplicação.
- `hashing.c`: arquivo onde as funções de espalhamento foram implementadas.
- `main.c`: arquivo que faz uso da biblioteca de funções hashing implementada e realiza os experimento descritos no relatório.
- Após a compilação e execução do arquivo `main.c`, alguns arquivos de texto são gerados no mesmo diretório contendo a resposta dos experimentos 1c, 2a e 2b.

## Como executar
- No ambiente Linux, é necessário primeiramente ter o compilador GCC instalado no sistema operacional. No Linux Ubuntu, por exemplo, o comando para instalação do compilador é: `sudo apt install gcc`.
- Após a instalação do compilador, é necessário acessar o diretório onde os arquivos do projeto se encontram e realizar o seguinte comando: `gcc main.c -o main`.
- Após a compilação do arquivo de código fonte e ainda no mesmo diretório, para executá-lo, é necessário realizar o seguinte comando: `./main`.
